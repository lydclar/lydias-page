School Simulator

As a project for one of my very first programming classes (an intro to C++) we were required to create a game. I came up with a school simulator in which the user could learn material, take tests, and move up to a higher level of that "class" if their test score is high enough. Upon completion of all courses, the user recieves a report card with their class grades listed, as well as a GPA.

Three.js Rotating Torus Checkerboard

I took a course on three.js to get more familiar with three dimensional programming. In order to demonstrate my skills, I created a page using React and Three.js displayig an alternating pink and green checkerboard pattern with a rotating torus in each square, complete with lighting and shadow techniques learned through the course.

Recreation of Asana Using Notion

The company I work for was using Asana for managing their project and task assignment, while they kept all project and task documentation in Notion. For a more cost effective and integrated solution, I was able to recreate the Asana dashboard within Notion to have everything in one, easy-to-access place. This meant that my job no longer required the use of Asana, and each of my coworkers were able to view their assigned tasks, documentation for their tasks, and even information about which website or client the task was for without the use of multiple applications.
